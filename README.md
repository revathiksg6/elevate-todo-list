# To-Do List Web App
This project is a simple front-end To-Do List Web App built using HTML, CSS, and Vanilla JavaScript.

## Objective
The objective of this project is to create a dynamic To-Do list where users can add, remove, and mark tasks as complete.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Instant UI updates without page reload

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. Enter a task in the input box.
2. Click the Add button or press Enter.
3. Click on a task to mark it as completed.
4. Click the Delete button to remove a task.

## Outcome
This project helped in understanding DOM manipulation, event listeners, and dynamic UI updates using Vanilla JavaScript.
